# WebSocket Fake para Azsim

## Sobre o Projeto

O WebSocket Fake para Azsim é uma ferramenta de simulação desenvolvida para gerar dados de teste que podem ser integrados com o sistema de monitoramento Azsim. Este projeto é ideal para desenvolvedores e testadores que precisam simular fluxos de dados em tempo real para verificar a funcionalidade e a eficiência do sistema Azsim.

## Funcionalidades

- **Geração de Dados Falsos:** Simula dados de eventos e ocorrências em um formato compatível com o Azsim.
- **Conexão WebSocket:** Utiliza o protocolo WebSocket para enviar dados simulados ao Azsim em tempo real.
- **Personalização de Dados:** Permite configurar os tipos e a frequência dos dados gerados para testes mais específicos.

## Tecnologias Utilizadas

- **JavaScript/Node.js:** Para o back-end do WebSocket e lógica de geração de dados.
- **Bibliotecas WebSocket:** Para estabelecer uma conexão em tempo real.
